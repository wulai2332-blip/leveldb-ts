leveldb-ts API Documentation

A TypeScript-native, zero-native-dependency, LevelDB-compatible embedded key-value storage engine with LSM-Tree architecture. Builds on Node.js (>=20) with pure TypeScript (ES2023, NodeNext). Package: @wulai2332/leveldb-ts — 25 core modules.

Runs in a worker_threads Worker. Handles two message types:

Worker Message Types

TypeKindFields

WorkerErrorinterface{ type: 'Error', message }

Compaction Algorithm (Sync & Worker)

1. Select input files from level N + overlapping files from level N+1
2. Open all SSTables and create iterators
3. Build a min-heap using InternalKey comparator (newest seq first)
4. K-way merge: emit the first occurrence of each unique InternalKey
   (deduplication — older versions of the same key are discarded)
5. Build output SSTable via TableBuilder
6. Write VersionEdit: delete old files, add new file
7. logAndApply to persist the change
